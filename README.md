# CadetCode: Python for Space Explorers

CadetCode is a web-based training platform designed to teach kids the basics of Python programming through a NASA-themed adventure.

## Current Missions
- **Sector 01: Text Module** (Printing, Strings, Multi-line art)
- **Sector 02: Math Sector** (Calculations, Variables as numbers)
- **Sector 03: Variable Sector** (Storage, input() functions, updating gauges)
- **Sector 04: Logic Sector** (if/else statements, comparisons)

## Features
- **NASA Telemetry:** Real-time Python execution using Pyodide.
- **Ghost Layer:** A visual guide that helps cadets type code accurately.
- **Save System:** Progress is tracked locally on the cadet's device.
- **Shield System:** Custom security layer (defense.js) to block ads and injections.

## Setup
1. Clone the repository.
2. Open index.html in any modern web browser.
3. No server or installation required.

## Security
The platform includes a built-in defense engine to maintain a clean, educational environment for young learners.
